# Security Policy

This repository is not open source. The source and issue reporting are for authorized personnel only.

## Supported Versions

| Version   | Supported          |
| --------- | ------------------ |
| 5.1.x     | :white_check_mark: |
| 5.0.x     | :x:                |
| 4.0.x     | :white_check_mark: |
| &lt; 4.0   | :x:                |

## Reporting a Vulnerability

This project is private. Do not open public issues or disclose vulnerabilities publicly.

Report security issues via the organization's internal ticketing system or to the internal Security Team. No external contact information is published.
